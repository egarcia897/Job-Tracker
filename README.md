Job Tracker

A simple and secure full-stack web application to help job seekers organize their career search. This app replaces messy spreadsheets with a clean, private dashboard to track every application from "Applied" to "Offer."
✨ Features

    User Accounts: Secure sign-up and login system.

    Private Dashboard: Each user only sees their own job applications.

    Job Management: Add, view, and delete job entries with ease.

    Status Tracking: Monitor progress with "Applied," "Interview," "Offer," and "Rejected" tags.

    Security: All passwords are encrypted and hashed for safety.

🛠️ Tech Stack

    Language: Python

    Framework: Flask

    Database: SQLite (SQLAlchemy)

    Styling: Bootstrap 5 (Pinterest-inspired UI)

📂 Project Structure

    app.py: The main logic and database setup.

    templates/: All HTML pages (Dashboard, Login, Register).

    jobs.db: The local database where info is stored.

🚀 How to Run Locally

    Clone the repo:
    Bash

    git clone https://github.com/egarcia897/Job-Tracker.git

    Install Flask:
    Bash

    pip install flask flask_sqlalchemy flask_login

    Start the app:
    Bash

    python app.py

    Open http://127.0.0.1:5000 in your browser.
