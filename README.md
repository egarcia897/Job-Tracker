# 🎯 Job Tracker App

A simple, secure web application to help job seekers track their applications in one place.

##  Key Features
* **Secure Login:** Create an account to keep your job list private.
* **Dashboard:** View all your active applications at a glance.
* **Track Status:** Mark jobs as *Applied*, *Interview*, *Offer*, or *Rejected*.
* **Mobile Friendly:** Built with Bootstrap for a clean, Pinterest-inspired look.

## 🛠️ Built With
* **Python & Flask** (The "Brain")
* **SQLite** (The "Database")
* **Bootstrap** (The "Style")

## 🏁 How to Use
1. Run `pip install -r requirements.txt`
2. Run `python app.py`
3. Open `http://127.0.0.1:5000` in your browser!
